Plus Ultra Futbol ⚽


Univerisyt Project | 2022

Summary:


The aim of this project was to create a Social Media and Digital project on a topic of your own choice which was to be completed as part of a final year project at university.

This project is an experimental Journalism and Graphic Design platform surrounding in-depth football analysis and topics – this project mainly focused on Financial Corruption in European football and female fan representation. This project allowed me to explore into geo-political football topics which are not widely discussed or given much focus too.

This topic was chosen as corruption in Football is always growing and are not discussed as frequently in-depth. This project provided me with an opportunity to explore these issues creatively.

Created a series of digital graphic design posters on Photoshop and Illustrator and coded a website with HTML, CSS and JavaScript in addition with the help of WIX. The project lasted 9 months in which all the content was created. The website enables the end user to access all information around the chosen topic – football and corruption. The project explored financial corruption within European football.

Examples such as Arab investors and their questionable political -activities, money laundering through clubs and also exploring female diversity in Football.



Built With:


HTML | CSS | SCSS | JavaScript | PhotoShop | Illustrator | InDesign
